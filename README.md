# MichaelCHurley.com

Personal resume, portfolio, and blog website for Michael C. Hurley.

## Overview

MichaelCHurley.com is a personal website showcasing professional experience, skills, portfolio projects, and blog content. Built with Next.js 14 and modern web technologies to provide a fast, responsive, and professional online presence.

## Technology Stack

- **Framework**: Next.js 14.2.8
- **Runtime**: React 18
- **Styling**: Tailwind CSS 3.4.1
- **UI Components**: Radix UI
- **Icons**: Lucide React, React Icons
- **Package Manager**: Bun
- **Hosting**: Vercel

## Features

- Professional resume display
- Portfolio showcase
- Blog functionality (planned)
- Responsive design
- Modern UI/UX
- SEO optimized

## Getting Started

### Prerequisites

- Bun installed (v1.3.1 or later)
- Node.js 18+ (if not using Bun)

### Installation

```bash
# Install dependencies
bun install

# Set up environment variables (if needed)
cp .env.example .env.local
```

### Development

```bash
# Run development server
bun run dev

# Open http://localhost:3000
```

### Building

```bash
# Build for production
bun run build

# Start production server
bun run start
```

### Linting

```bash
bun run lint
```
